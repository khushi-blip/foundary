# 🚀 Foundry – AI-Powered Dataset Generation Platform

🧠 Foundry is an AI-powered platform that autonomously creates, mines, and annotates datasets using intelligent multi-agent systems. It enables automated image collection, object detection, annotation, and dataset export in COCO format, significantly reducing manual labeling effort.

Built with 🤖 Google Gemini and a 🧩 multi-agent architecture, Foundry streamlines end-to-end dataset creation using natural language commands and intelligent automation.

## 🧠 Overview

This project was developed collaboratively as part of an intensive AI program.  
My contributions focused on documentation, setup, and team collaboration, ensuring smooth integration of features and system workflows.

---

## ⚙️ Core Capabilities

Using a multi-agent system powered by Google Gemini, Foundry can:

📥 Collect and curate relevant images intelligently  
🔍 Detect multiple objects within images  
🏷️ Generate accurate bounding box annotations automatically  
🤖 Filter low-quality or irrelevant data using AI validation  
🧾 Export datasets in COCO format seamlessly  
💬 Enable natural language-based dataset creation  
🔁 Retry and self-correct failed or low-confidence annotations
