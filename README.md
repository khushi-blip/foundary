#Foundary

Foundry is an AI-powered platform that automatically creates objects and detects datasets using intelligent AI agents. It supports searching, filtering, annotation, and exporting datasets in COCO format autonomously.

Built using Google Gemini and a multi-agent architecture.
This project was developed collaboratively as part of an intensive AI project, where I contributed to documentation, setup, and project collaboration.

The system uses a multi-agent architecture with Google Gemini integration to
* Collect and filter images intelligently
* Detect and annotate multiple objects
* Generate bounding boxes automatically
* Export datasets in COCO format
* Support natural language dataset creation

## Features

* AI-driven dataset mining
* Multi-object annotation support
* Natural language interaction
* Automated quality filtering
* COCO dataset export
* Retry and validation mechanisms
* Custom image annotation support

## Tech Stack

* Python
* Google Gemini API
* Multi-Agent Architecture
* Computer Vision
* COCO Dataset Format




## Project Goal

The goal of Foundry is to simplify dataset generation and reduce manual annotation effort using autonomous AI agents.

Developed collaboratively during an intensive AI project program.
